# RoboViz-Arm-V0.2

## Description
A robotic manipulator using ROS2. Implemented Robot Kinematics, Trajectory Planning with MoveIt2, and the ROS2 control library. Integrated voice-activated control through Alexa. 

### Applications Used
* ROS 2
* Gazebo 
* MoveIt! 2
* RViz 2
